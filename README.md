# AutoCAD PDF Reader

## How to Run

pip install -r requirements.txt
set GROK\_API\_KEY=your-key

set OPENAI\_API\_KEY=your-key
python drawingSystem.py
python drawingSystem.py



\## Environment Setup



Before running the program, set your API keys:



\### Windows

setx OPENAI\_API\_KEY your-openai-key

setx GROK\_API\_KEY your-grok-key



\### macOS/Linux

export OPENAI\_API\_KEY=your-openai-key

export GROK\_API\_KEY=your-grok-key
